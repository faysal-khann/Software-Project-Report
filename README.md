# Student Skill Bridge

## Software Engineering Project Report

Student Skill Bridge is a software engineering project developed to create a bridge between students and employers by providing a centralized platform for internships, part-time jobs, and recruitment opportunities. The system enables students to build professional profiles, upload resumes, apply for jobs, and communicate with employers, while employers can post jobs, review applications, and schedule interviews.

---

# Project Objectives

- Connect students with employers efficiently
- Simplify internship and job application processes
- Provide AI-based job recommendations
- Enable secure communication between students and employers
- Improve recruitment and application management
